# Grupo 4 Predictor de Vida Útil de Tanques (Faraday)
Software - Predictor de Vida Útil de Tanques (Faraday)

Mediante este software se busca predecir la vida útil remanente de un tanque metálico sometido a corrosión, utilizando parámetros geométricos y velocidades de corrosión estimadas para distintos ambientes.
